My Projects

Decentralized Mesh Networks for Disaster Resilient Connectivity

This project develops a decentralized wireless mesh network using Raspberry Pi nodes, LoRaWAN, Wi-Fi 6, and TV White Space (TVWS) to provide resilient communication in flood-prone areas like the Tana River Delta. The system self-heals, supports peer-to-peer connectivity, and prioritizes emergency traffic, solving the problem of connectivity loss during disasters and promoting last-mile digital inclusion.
